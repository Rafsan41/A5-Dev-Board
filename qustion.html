Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML:- it reads both the HTML markup and the text content of the element. innerText:- it returns the text as it appears on screen. It ignores HTML tags textContent:- it returns only the text. Whiles innerText reads text as it is rendered on screen
Question-3: What is event delegation in the DOM?
JavaScript event delegation is a simple technique by which we are adding a single event handler to a parent element in order to avoid having to add event handlers to multiple child elements.
Question-4: What is event bubbling in the DOM?
Event bubbling follows the opposite order as event capturing. An event propagates from a child HTML element, then moves up the DOM hierarchy to its parent elements: Event bubbling hierarchy = child → parent → body → html → document Listening to propagation events
